ngRainbow.js

Introduction

This is a Rainbow.js plug-in for AngularJS and most code is used from Rainbow.js. Kudos to its author: Buu Nguyen.

Demo

Usages

Installation

Download ngRainbow.js and include it in the page: <script src="ngRainbow.js"></script>
Include the corresponding module in your module: var app = angular.module('ngRainbow', []);
Import the Rainbow.js provider in the controller: app.controller('Controller', ['rainbowBar', function(rainbowBar) {}])

API
Show

rainbowBar.show();

Hide

rainbowBar.hide();

Configuration

Use the corresponding provider to configure the rainbowBar before using it.
Options available(Quoted from Rainbow.js):

  • autoRun (default: true): if false, call rainbow.progress() manually to run.
  • barThickness (default: 3): the progress bar thickness.
  • barColors: list of gradient color stops used to draw the progress bar.
  • shadowBlur (default: 10): the shadow blur.
  • shadowColor: the shadow color.
app.config(['rainbowBarProvider', function(rainbowBarProvider) {
    rainbowBarProvider.configure({
        barThickness: 5
        shadowColor: 'rgba(0, 0, 0, .5)'
});}]);
